# Metrics to Show a Buyer

## Content Ops
- Total published articles
- Publishing cadence (7/30-day trend)
- Rejected auto rate and top QA flags
- Category coverage depth

## Technical Health
- Workflow success rate by stage
- Average time topic -> publish
- Budget burn rate vs quota target
- Revalidate success rate

## SEO / Traffic Readiness
- Indexed pages (Search Console)
- Impressions / CTR trends
- Core page performance (LCP/CLS)
- Internal link coverage

## Monetization Readiness
- Ad slot placements already integrated (feature-flagged)
- Legal/privacy/cookie pages present
- Analytics instrumentation ready
