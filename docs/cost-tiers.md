# Cost Tiers (Estimated)

## Low (< $100/month) - Current MVP target
- VPS: low-cost single instance
- Vercel: free or low tier
- Sanity: free/entry usage
- AI text/image: throttled publish quota (4-8/day)
- Strategy: many topic candidates, selective publish

## Balanced ($100-$300/month)
- Better text QA model usage and retries
- More reliable image generation throughput
- Higher publish quota (8-15/day)
- Optional monitoring and backups upgrades

## Scale-ready (> $300/month)
- Higher throughput workflows, richer QA checks
- More images/variants per article
- Stronger observability, staging env, premium infra
- Can support burst mode closer to 20+/day published
