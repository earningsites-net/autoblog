# Buyer Handoff Checklist

## Accounts / Access
- Vercel project (production + env vars)
- Sanity project (owner/admin transfer)
- VPS provider account or server credentials
- n8n editor credentials and encryption key
- AI provider accounts / keys
- Analytics (GA4) and Search Console properties
- Domain registrar and DNS

## Technical Assets
- Git repository (this repo)
- n8n exported workflows
- `.env` templates and current key mapping (without secrets in plaintext email)
- Runbook and SOP docs
- Cost model and current monthly spend estimate

## Verification Before Transfer
- New article can be generated and published end-to-end
- Frontend revalidate works
- Sitemap includes recent articles
- Legal pages exist and contact details are updated
- Ad placeholders/flags documented (enabled vs disabled)
